**Title:** SEO Gospel Zero
**Version:** 1.0
**Purpose:** Foundation rules for how VR Creative Group approaches SEO.

* Start with user intent.
* Build pages for humans first, algorithms second.
* Keep structure simple, repeatable, and scalable.
* Every page must load fast, read clean, and connect to a tag structure.
* All content must be useful, direct, and aligned with the performance product.
* Visuals must support clarity and conversion.
