# Overview
This repository contains the complete structure of the VR Creative Group technical ecosystem. Every folder mirrors a core domain of the business:

- `/code` – Loaders, HTML, CSS, JS, Apps Script
- `/seo` – Gospels, metadata, schemas, redirects
- `/brand` – Identity, logos, typography, merch
- `/prompts` – Master prompts for every system
- `/systems` – All operational engines
- `/notes` – Strategy, show architecture, internal documentation
- `/youtube` – Metadata, thumbnail styles, export scripts
- `/deployment` – Hosting, cloud setup, packaging

All subfolders contain structured markdown files to guide development and implementation.

# Usage Guide

1. Open a folder to view its markdown assets.  
2. Use `/code` when editing site code or Apps Script.  
3. Use `/systems` when modifying backend logic.  
4. Use `/seo` for any metadata, tag, schema, or SEO revisions.  
5. Use `/brand` for visual identity updates.  
6. Use `/notes` for strategic direction and show development.  
7. Use `/deployment` when shipping or hosting apps.  
8. Use `/prompts` to build, regenerate, or iterate systems with AI.
