About SEO Codex

SEO Codex is a show management system for VR Creative Group. It helps you manage your entertainment show database with proper SEO optimization.

‚ú® Features

  • Load show data from multiple sources (URLs, local files, Google Sheets)
  • Browse and search your entire show catalog
  • Filter shows by tags for easy discovery
  • SEO-optimized metadata for each show
  • Responsive design that works on all devices

üöÄ Quick Start

  1. Go to the "Load Data" tab
  2. Choose a loading method (URL, file, or sample data)
  3. Browse your shows in the "Browse All Shows" tab
  4. Use "Filter by Tag" to find shows by category

üìä Data Format

Your JSON file should follow this structure:

{
  "shows": [
    {
      "title": "Show Name",
      "meta_title": "SEO Title",
      "meta_description": "SEO description...",
      "page_slug": "show-name",
      "tags": ["tag1", "tag2"],
      "imageurl": "https://...",
      "videourl": "https://...",
      ...
    }
  ]
}

üí° Tips

  • Use the sample data to test before loading your own
  • Ensure your JSON follows the required format
  • Tags help with organization and SEO
  • All image URLs should use HTTPS
