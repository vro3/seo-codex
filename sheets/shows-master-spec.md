**Title:** Shows_Master Sheet Specification
**Purpose:** Defines mandatory columns.

Required columns:

* title
* tagline
* short_description
* full_bio
* event_type
* experience_type
* tags
* page_slug
* meta_title
* meta_description
* imageurl
* imageurl2
* imageurl3
* videourl
* previewvideourl
* instrumentation
* group_size
* setup
* tech
* variant
* related_tags

Rules:

* Slugs must be unique.
* Tags must be comma-separated lowercase.
* Metadata must always be filled.
* No empty images.
