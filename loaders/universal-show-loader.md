**Title:** Universal Show Loader v1.5
**Purpose:** Defines the show page system.
**Note:** This file describes the system, not the code.

* Pulls full show data from Shows_Master.
* Builds hero image, description, metadata, related shows.
* Autolinks tag buttons to filtered pages.
* Injects SEO: meta title, meta description, OG tags.
* Outputs clean HTML for Squarespace.
* Requires fields:

  * title, tagline, full_bio, tags, imageurl, videourl, page_slug.
